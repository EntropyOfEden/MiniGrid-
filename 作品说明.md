# 作品说明文档（深度强化学习大作业）

## （1）作品功能与应用场景

**功能**：默认在 Farama **MiniGrid** 环境 **`MiniGrid-ObstructedMaze-2Q-v1`** 中训练智能体：在带门锁与障碍的 3×3 房间迷宫中完成指定任务（拾起目标球；环境中会以自然语言任务提示）。观测为**全局可见**的 RGB 栅格图像（经封装为神经网络输入），动作空间为 MiniGrid 默认离散动作（转向、前进、`pickup`、`toggle` 等）。训练完成后可对策略进行贪心评估，并支持可视化窗口演示。也可通过 `--env-id` 切换到更简单的地图（如空房间）做对照实验。

**应用场景**：网格地图上的自主导航与路径规划可作为简化原型，对应仓储机器人分拣路径、清扫机器人区域覆盖、游戏 NPC 寻路等任务的抽象；本作业侧重算法验证与小规模仿真演示。

## （2）使用的强化学习算法与训练方式

### 算法

- **DQN（Deep Q-Network）**：用卷积神经网络近似动作价值函数 \(Q(s,a)\)，以 \(\epsilon\)-贪婪策略采集样本，使用**经验回放**与**目标网络**缓解样本相关性与训练不稳定。
- **Double DQN（作为改进）**：在计算 Bootstrap 目标时，用**在线网络**选择下一动作、用**目标网络**评估该动作的 Q 值，减轻标准 Q 学习中的**过高估计（overestimation）**问题。

公式要点（省略括号中的状态依赖）：

- TD 目标（DQN）：\(y = r + \gamma \max_{a'} Q_{\theta^-}(s', a')\)
- TD 目标（Double DQN）：\(y = r + \gamma Q_{\theta^-}(s', \arg\max_{a'} Q_{\theta}(s', a'))\)

其中 \(\theta\) 为在线网络参数，\(\theta^-\) 为目标网络参数（周期性从 \(\theta\) 复制）。

### 训练流程（与本仓库代码对应）

1. **交互采样**：每个环境步根据 \(\epsilon\)-贪婪选择动作，将转移 \((s,a,r,s',\text{done})\) 存入回放缓冲区。
2. **随机小批量更新**：从缓冲区随机采样批次，最小化 Huber（Smooth L1）形式的 TD 误差。
3. **目标网络同步**：每隔固定环境步数将在线网络参数复制到目标网络。
4. **探索退火**：\(\epsilon\) 从初值线性衰减至终值，逐步从探索过渡到利用。

### 额外改进（工程层面）

- **每步小额惩罚**（奖励整形）：在稀疏奖励基础上增加微弱负奖励，鼓励更短路径；需在文档/视频中说明其可能影响最优策略与仿真假设。
- **卷积编码器 + 全连接头**：将图像观测映射为各动作的 Q 值向量。

超参数见 `train.py` 命令行默认值（学习率、折扣因子、批次大小、回放容量、目标网络更新间隔等）。

## 附录：改进点详情：
1. Double DQN（相对 Vanilla DQN）

- **做法**：在算 Bootstrap 目标时，用**在线网络**选 \(\arg\max_a Q(s',a)\)，用**目标网络**估 \(Q(s',\cdot)\)，减轻 **Q 值过估计**。
- **位置**：`src/dqn_agent.py` 里 `use_double_dqn` 分支；训练时加 **`--double-dqn`**。
- **对比实验**：`scripts/compare_double_dqn.py` 会先后训练「不加 / 加 Double DQN」并画 `runs/compare_dqn_double.png`。

2. 每步奖励整形（Reward shaping）

- **做法**：在 `src/env_setup.py` 的 `StepPenaltyRewardWrapper` 里，对环境每步奖励加一个 **`step_penalty`**（默认 `-0.001`），鼓励更短轨迹；环境自带的稀疏成功奖励仍在。
- **训练参数**：`train.py` 的 **`--step-penalty`**（默认 `-0.001`）。

3. 探索调度

- **做法**：\(\epsilon\)-贪婪从 **`--eps-start`** 线性降到 **`--eps-end`**，衰减长度由 **`--eps-decay-steps`**（环境步数）控制。
- **位置**：`train.py` 里的 `epsilon_by_step`。
